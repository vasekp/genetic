Detailed Description

The main namespace of the framework.

The entry point of most applications is either gen::Population or gen::NSGAPopulation. Other ...Population classes are not intended to be used directly.

Variable Documentation

thread_local std::minstd_rand gen::rng {std::random_device{}()}
static

The default random number generator for this framework.

A strong RNG is not a necessity for genetic applications, so speed was main preference in choosing std::minstd_rand. Can be accessed freely by applications of the framework. Similarly, alternative RNGs may be provided to functions like randomSelect().
